Windows实战Git环境配置msysGit+TortoiseGit

Git是一个分布式源代码版本管理控制程序. “Git 是一个由林纳斯·托瓦兹为了更好地管理linux内核开发而创立的分布式版本控制／软件配置管理软件。”- 维基百科中文。http://zh.wikipedia.org/zh-cn/Git

 

msysgit和TortoiseGit的开发进展很快，我们在windows平台上使用的git，越来越强大了。虽然相比Linux下面的GIT还有很大差距，但相比其它版本控制系统，git 还是可以 笑傲江湖。

 

它有如下优点：

1. 它足够稳定，linux的核心就是用它来托管的。数G大的源码仓库，依然稳定如山。

2. 它足够快，是所有版本控制程序中最快的一个。

3. 它被人们所高度关注，有很多人为它写书，出教程。proGit就是其中很好的一本书。

4. 易学易用，几乎不存在门槛。

5. 无需花费成本，它是开源免费软件。

6. 跨平台，所有熟知的平台下面，都跑得很欢畅。

在介绍了一堆优点之后，让我们来进入Git的世界吧。linuxers 们，你们就不用看这里了。因为你们git骄子。linux下面铁定好使。windows下面的用户就需要自己动手，打造可用的git环境了。

感谢msysGit(http://code.google.com/p/msysgit/)和tortoiseGit(http://code.google.com/p/tortoisegit/)项目，让我们可以一次性装好所需的软件，而不用纯手动编译安装Git.

 

下载msysgit 和tortoiseGit

msysGit:http://code.google.com/p/msysgit/downloads/list

TortoiseGit: http://code.google.com/p/tortoisegit/downloads/list

下载后先安装TortoiseGit， 然后安装msysgit.

装上它们之后。可能需要重启一下系统。等windows重启后，你的电脑就可以使用git了。

在桌面或任意位置，点击鼠标右键，就应该可以看到TortoiseGit的菜单了。

尽情享用Git吧！

 

附上eclipse的egit插件安装方法：

使用Market Place安装EGit

Eclipse 3.6的用户透过Eclipse上的 EclipseMarketplace 就可以安装EGit。 3.6以下版本的安装网址请参考http://www.eclipse.org/egit/download/。

a. 开启Eclipse后请选择Help→Eclipse Marketplace；

b. 请选择Eclipse Marketplace，按Next >继续；

c. 在Find字段键入egit再按下Go按钮开始搜寻；

d.搜寻完后会显示如下结果，请按下Install按钮开始安装；

e.此画面显示安装的明细，请按Next >按钮继续；

f.请选择I accept the terms of the license agreement，按下Finish按钮开始安装；

g.开始进行安装；

h.安装完成后请选择Restart Now，重新开启Eclipse安装的EGit才可使用。

装上它们之后。可能需要重启一下系统。等windows重启后，你的电脑就可以使用git了。

 

更多参考：http://xiaogui.org/windows-git-with-msysgit-and-tortoisegit.html
